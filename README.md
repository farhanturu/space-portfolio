# Stellar Portfolio

A cosmic-themed portfolio website featuring immersive 3D space scenes built with React Three Fiber and Next.js.

## Features

- Interactive 3D space environment with floating geometries and particles
- Smooth scroll animations powered by Framer Motion
- Responsive design for all devices
- Dark mode space aesthetic
- Contact form with validation

## Tech Stack

- **Framework:** Next.js 16
- **3D Engine:** Three.js + React Three Fiber + Drei
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion
- **Icons:** Phosphor Icons
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Build

```bash
npm run build
npm start
```
